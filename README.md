# Atividade IV Clinica Vida-Feliz
Atividade de Modelagem de Banco de Dados com script funcional

Banco de dados relacional para gerenciamento de pacientes, agendas e consultas.

## Como usar
1. Abra o MySQL Workbench ou outro client MySQL.  
2. Conecte ao servidor MySQL.  
3. Execute o arquivo `atividadeivclinicavidafeliz.sql`.  
4. As tabelas, inserções, consultas, atualizações e exclusões já estão incluídas no script.
